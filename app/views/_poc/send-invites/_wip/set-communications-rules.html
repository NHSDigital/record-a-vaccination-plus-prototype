Invite rules

Set rules to determine when patients are sent different vaccine invites.

The send invites feature is designed to be used routinely - for example, once a week or once a fortnight.

This screen shows the rules that determines which type of invite (if any) a patient will be sent. There are two types of rules:

  • rules relating to when a targeted invite should be sent (Pertussis, RSV, Seasonal or 'Mop up')
  • rules relating to how often a patient should be communicated with if they still have outstanding vaccinations (upper and lower limits)

In all instances, when there is more than one instance of a message being scheduled, these messages will not be sent if the patient has already had the vaccine in question OR has made a booking to receive it.
