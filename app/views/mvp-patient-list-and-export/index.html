Patient list and export

Description

Facilitate a logged in user to view their successfully uploaded and enriched patient list.

The patient list allows for filtering to be applied, as well as issues being noted with problematic patient records that were part of the enrichment process.

Enriched patient list row level errors will be played back to the logged in user in a different view to the successfully enriched patient list.

As part of the patient view, the user will also be able to export the enriched data with, or without, filters applied.

Entry point

  1. Maternity landing page

View patient list

  1. No filters
  2. Filters applied
  3. Row level errors

Export patient list

  1. Export (no filters)
  2. Export (filters applied)
  3. Export completed (no filters)
  4. Export completed (filters applied)
